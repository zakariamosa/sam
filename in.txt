
After pushing successfully to GitHub, on the Mac do this step-by-step.

1. Install Homebrew

Open Terminal:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Verify:

brew --version
2. Install Python + Tesseract + Git
brew install python@3.12
brew install tesseract
brew install git

Verify:

python3 --version
tesseract --version
git --version
3. Clone your repo

Go somewhere you want:

cd ~

Clone:

git clone https://github.com/zakariamosa/sam.git

Enter folder:

cd sam
4. Download the SAM model

Create models folder:

mkdir models

Download model:

curl -L https://dl.fbaipublicfiles.com/segment_anything/sam_vit_b_01ec64.pth -o models/sam_vit_b_01ec64.pth

This may take a little time.

5. Create virtual environment
python3 -m venv .venv

Activate it:

source .venv/bin/activate

You should see:

(.venv)
6. Upgrade pip
pip install --upgrade pip setuptools wheel
7. Install PyTorch CPU

Intel Mac:

pip install torch torchvision
8. Install project dependencies
pip install -r requirements.txt
9. Set environment variables
export SAM_API_KEY="change-me"
export SAM_DEVICE="cpu"
10. Start the server
python -m uvicorn app:app --host 0.0.0.0 --port 8000

If successful you should see something like:

Application startup complete
Uvicorn running on http://0.0.0.0:8000
11. Test health endpoint

Open another terminal window:

curl http://127.0.0.1:8000/health

You should get JSON response.

12. Test detection
curl -X POST "http://127.0.0.1:8000/detect-tickets" \
  -H "x-api-key: change-me" \
  -F "image=@/full/path/to/test.jpeg"

Now your Mac mini is officially a SAM inference server.
