Earlier I included Tesseract because your old script had OCR fields like:

ocr_confidence
ocr_word_count

But after we changed the direction, your SAM service does not need OCR anymore.

For your real project, the Mac/server should only do:

image → SAM polygons → JSON

So you can skip Tesseract completely.

Install only:

python3
git

Then inside the project:

python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip setuptools wheel
pip install torch torchvision
pip install -r requirements.txt

But make sure requirements.txt does not force Tesseract/OCR dependencies like:

pytesseract

If it does, remove pytesseract from requirements.txt for the server version.
